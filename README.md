# gofiber-cli

A monorepo containing CLI tools for Go Fiber development.

## Tools

### Optogics (`optics`)
modular-based CLI for generating Go Fiber backends.

Install:
```bash
go install github.com/jemiezler/gofiber-cli/optics@latest
```

Usage:
```bash
optics <command>
```

Commands:
```bash
optics new <project-name>
```

```bash
optics g resource <module-name>
```
